xform

NAME

default - Transform JSON data from stdin, or as specified by [&lttemplate-file&gt]

SYNOPSIS

xform [&lttemplate-file&gt] [--vars=&ltname:value[,...]&gt][--out=&ltoutput-file&gt]

DESCRIPTION

Transform JSON data as specified by json-xform:

JSON data is the contents of &lttemplate-file&gt if provided or the contents of stdin if &lttemplate-file&gt otherwise.

If --out=&ltoutput-file&gt was not provided; writes the output to stdout, otherwise overwrites/creates without prompt the file specified with transformed contents.

OPTIONS

-V=&ltname:value[,...]&gt

--vars=&ltname:value[,...]&gt

Override or create the root @xform:var graph, in memory, on the parsed result of [&lttemplate-file&gt] or stdin.

echo {"test":"${example}"} | xform --vars=example:"Mic Check"

-o=&ltoutput-file&gt

--out=&ltoutput-file&gt

Override or create the root @xform:var graph, in memory, on the parsed result of [&lttemplate-file&gt] or stdin.
